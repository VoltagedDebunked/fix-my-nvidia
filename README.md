# fix-my-nvidia
Nvidia drivers suck on linux. so this fixes em.

## How to use

1. Clone the repository:
```bash
git clone https://github.com/voltageddebunked/fix-my-nvidia.git
cd fix-my-nvidia
```

2. Make the script executable:
```bash
chmod +x script.sh
```

3. Run:
```bash
sudo ./script.sh
```

Enjoy!
